# JDBC

## 知识点

* 各大数据库厂商提供的驱动实现了JDBC接口规范
* 四大核心对象：DriverManager、Connection、Statement和ResultSet
* 使用JDBC开发的步骤
  * 注册驱动
  * 获得连接
  * 获得SQL语句对象
  * 执行SQL语句获得结果
  * 处理结果
  * 释放资源
* 使用JDBC进行各种增删查改操作
* 如何使用PreparedStatement来避免SQL注入
